Moderator Commands









ban


Bans mentioned member of your server




clean


Clean recent messages by the bot




delwarn


Delete a warning from member of your server




kick


Kick mentioned member from your server




lock


Locks a channel so no one can type




modlogs


Get all recent modlogs done on a user by a moderator




mute


Mute a user so they cannot type




nickname


Set the nickname of a user with just a command




setmodlogs


Set the modlogs channel for the server




softban


Softban a user from your server




unban


Unban a user who is banned already




unlock


Unlocks a channel which is loacked already




warn


Warn a user if you notice misactivity




warnings


Check the warnings corresponding to a user
